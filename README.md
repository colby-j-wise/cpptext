## Requirements and Dependencies

- C++ 14
- Threading library (pthread, ...)
- GNU Scientific Library (GSL)
- OpenMP

## Building

Building and running is straightforward:

`./build.sh && ./run.sh`

To clean do:

`./clean.sh`

To delete all build files and binaries do:

`./distclean.sh`
